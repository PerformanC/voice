# getSpeakStream 

## Description

Gets the stream of what an user is saying in the connected voice channel.

## Syntax

```
perfcVoice.getSpeakStream(ssrc)
```

## Parameters

- `ssrc` - The SSRC of the user.

## Return value

If user is found, a `PassThrough` class instance. Otherwise, `null`.
